encriptarContrasena (   $password)

Metodo de seguridad. Encripta la contraseña mediante el algoritmo SHA1. Todas las validaciones y almacenamientos se hacen en este sistema. Las bases de datos no guardaran contraseñas tal cual las incluye el usuario.

Parameters
$password- Contraseña a encriptar
Returns
contraseña encriptada en SHA1

Metodo que se encarga de guardar en la base de datos una contraseña nueva Cuando el usuario la ha olvidado. Este metodo solo puede ser accedido desde el mensaje recibido en el correo

$correo- Correo electronico del usuario que cambia la contraseña
$validacion- Valor aleatorio para verificar que el usuario realmente recibió el correo
